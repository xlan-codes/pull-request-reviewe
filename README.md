# AI-Agentic Pull Request Reviewer

A fully autonomous multi-agent AI system for reviewing Pull Requests and Merge Requests across GitHub, GitLab, and Bitbucket.  
It combines **RAG**, **multi-agent orchestration** to produce high-quality, actionable code reviews.

---

# Purpose

The goal of this project is to build an intelligent, automated PR/MR reviewing system that:

- Identifies code issues in real-time  
- Understands context using RAG (Retrieval-Augmented Generation)  
- Applies coding standards, best practices, and security rules  
- Uses multiple specialized AI agents to improve accuracy  
- Self-corrects using reflection to reduce false positives  
- Supports GitHub, GitLab, and Bitbucket uniformly  
- Provides a clean UI using Streamlit  

This system demonstrates a **production-grade AI-Agentic architecture** for enterprise code review automation.

# Quick Start

## Prerequisites
- Python **3.10+**
- **OpenAI API key**
- (Optional) GitHub/GitLab/Bitbucket tokens for private repositories

---

# Installation

### 1. Navigate to the project
```bash
cd /Users/xlan-codes/Desktop/personal_projects/rag/pull-request-reviewer
```

### 2. Create virtual environment
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Configure environment variables
```bash
cp .env.example .env
```

Edit `.env` and add:

```env
OPENAI_API_KEY=your_key_here

# Optional tokens for private repos
GITHUB_TOKEN=
GITLAB_TOKEN=
BITBUCKET_USERNAME=
BITBUCKET_PASSWORD=
```

# Usage

### Using the UI
- Enter a GitHub/GitLab/Bitbucket PR URL  
- Select review mode: *quick, standard, deep*  
- View categorized feedback generated by multi-agents  

### Using CLI
```bash
python scripts/run_demo.py --url https://github.com/owner/repo/pull/123
```

### Programmatically
```python
from src.agents.crew_orchestrator import ReviewCrew

crew = ReviewCrew()
result = crew.review_pull_request(pr_url="https://github.com/owner/repo/pull/123")
print(result.feedback)
```
